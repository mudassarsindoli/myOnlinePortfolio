noDice

noDice is a small project designed and developed by me. This is my very first project & I have created it using raw HTML, CSS & JavaScript. The goal of this project is to eliminate the use of dice while playing board games. You can just open this web app on your phone/laptop/desktop and start playing right away! Using this concept, "waiting for your turn" to roll the dice is eliminated. All the 4 members can roll the dice at once and play simultenously which will keep the excitement throughout the game.
